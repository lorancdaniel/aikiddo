# AI Kids Music Studio

Lokalna aplikacja do produkcji piosenek i animowanych klipow dla dzieci z uzyciem modeli AI uruchamianych na wlasnym sprzecie.

Glowny plan projektu znajduje sie tutaj:

- `docs/superpowers/plans/2026-04-25-ai-kids-music-studio.md`

Katalogi robocze:

- `app/` - przyszly kod aplikacji, API, workerow i pipeline'ow
- `projects/` - lokalne projekty produkcyjne, artefakty audio/video i eksporty
- `research/` - notatki o modelach, testach jakosci i benchmarkach
- `docs/` - dokumentacja techniczna i plany
